# 要件定義

**プロジェクト名**: キッズTodoアプリ
**作成日**: 2025年12月7日
**対象ユーザー**: 小学校2年生（7-8歳児）

---

## 📋 プロジェクト概要

小学校2年生向けの可愛く親しみやすいタスク管理アプリ。
毎日同じタスクをベースに日々のチェックができ、タスクの追加・完了・削除が簡単に行える。
ふわふわ煙の「ふわこ」がメインキャラクターとなり、宿題を応援してくれる。

### 目的
- 子どもたちが自分でやることを管理できるようになる
- タスク完了の達成感を味わい、自己肯定感を高める
- 楽しく習慣づくりをサポートする

### 解決したい課題
- 子どもが毎日やることを忘れてしまう
- やることリストを見るのが面倒・つまらない
- 達成感が得られにくく、モチベーションが続かない

---

## 👤 ユーザーストーリー

### US-1: 毎日のタスクを確認する
**As a** 小学2年生  
**I want to** 今日やることを見る  
**So that** 何をすればいいか分かる

**受け入れ基準**:
- [ ] 今日の日付が表示される
- [ ] 今日やるタスクのリストが表示される
- [ ] タスクは大きな文字で読みやすく、小学校1年までで習う常用漢字とひらがな、カタカナのみで構成されている
- [ ] 可愛いイラストやアイコンがある

---

### US-2: タスクを完了する
**As a** 小学2年生  
**I want to** やったことにチェックをつける  
**So that** 自分が頑張ったことが分かる

**受け入れ基準**:
- [ ] タスクをタップ/クリックすると完了できる
- [ ] 完了したタスクには分かりやすいマークがつく
- [ ] 完了時に嬉しいアニメーションや音が出る
- [ ] 「すごい！」「やったね！」などの褒めメッセージが表示される

---

### US-3: 新しいタスクを追加する
**As a** 小学2年生  
**I want to** 自分でやることを追加できる  
**So that** 忘れたくないことを覚えておける

**受け入れ基準**:
- [ ] 簡単な操作でタスク追加画面を開ける
- [ ] ひらがな・カタカナ・漢字で入力できる
- [ ] 絵文字やアイコンを選べる（オプション）
- [ ] 追加ボタンが大きくて押しやすい

---

### US-4: タスクを削除する
**As a** 小学2年生  
**I want to** いらなくなったタスクを消す  
**So that** リストがスッキリする

**受け入れ基準**:
- [ ] タスクを長押し（またはボタン）で削除できる
- [ ] 「本当に消す？」と確認メッセージが出る
- [ ] 確認メッセージも子ども向けの優しい言葉

---

### US-5: 毎日のタスクテンプレートを管理する
**As a** 小学2年生（または保護者）  
**I want to** 毎日繰り返すタスクを登録しておく  
**So that** 毎日同じことを入力しなくていい

**受け入れ基準**:
- [ ] よく使うタスク（歯磨き、宿題など）をテンプレートとして保存できる
- [ ] テンプレートから毎日のタスクが自動生成される
- [ ] テンプレートの編集・追加・削除ができる

---

## 🎯 主要機能

### 1. タスク表示機能
- 今日の日付表示（分かりやすい形式）
- タスクリスト表示
- 完了/未完了の視覚的な区別
- 大きな文字とアイコン

### 2. タスク完了機能
- タップ/クリックで完了切り替え
- 完了時の祝福アニメーション
  - キラキラエフェクト
  - ポジティブな音
  - 褒めメッセージ（ランダム）
- 進捗表示（例: 「5こ中3こできたよ！」）

### 3. タスク追加機能
- 簡単な入力フォーム
- アイコン/絵文字選択（オプション）
- 即座に反映

### 4. タスク削除機能
- 分かりやすい削除UI
- 確認ダイアログ
- 間違えて消さない配慮

### 5. テンプレート管理機能
- 毎日繰り返すタスクの登録
- テンプレートからの日次タスク生成
- テンプレートの編集

### 6. データ永続化
- ローカルストレージでの保存
- 日付ごとのタスク状態保存

## 🎨 UI/UX要件

### デザイン方針
- **可愛らしさ**: 丸みを帯びたデザイン、パステルカラー
- **分かりやすさ**: 大きなボタン、シンプルな構成
- **楽しさ**: アニメーション、イラスト、褒め言葉

### カラーパレット
- 明るく優しい色合い
- コントラストは十分に（読みやすさ重視）
- 子どもが好む色（たまごっちのサイトhttps://tamagotchi-official.com/jp/を参照）

### フォント
- 大きめの文字サイズ（16px以上）
- 読みやすいゴシック体
- ひらがなが読みやすいフォント

### アイコン・イラスト
- 分かりやすいアイコン（星、ハート、ユニコーンなど）
- 可愛いキャラクター（オプション）
- 全件完了時の祝福エフェクト（クラッカーが弾けて星や花吹雪が飛ぶ）、「さあ、あそぶじかんだ！」などのランダムメッセージ
